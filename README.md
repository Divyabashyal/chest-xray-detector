#  Chest X-Ray Pneumonia Detector

AI powered chest X-ray pneumonia detection using ResNet18 and GradCAM explainability.

## Features
- Binary classification: Normal vs Pneumonia
- GradCAM attention maps showing AI decision areas
- Built with PyTorch and Python

## Live Demo
https://huggingface.co/spaces/divyaml/chest-xray-detector

## Tech Stack
- PyTorch
- ResNet18 (Transfer Learning)
- GradCAM
- Gradio
- Python
